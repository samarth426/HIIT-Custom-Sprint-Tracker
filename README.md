## ⚡ Core Engineering Features
* **Dynamic Inputs:** Allows the user to fully customize total sprints, sprint durations, and rest times.
* **Dual-Phase Logic:** Uses nested loop architectures to run separate countdown tracking mechanisms for both running and active recovery phases.
* **Edge-Case Conditioning:** Implements smart programmatic checks to automatically skip the rest countdown on the final round.

## 🛠️ How to Run
Ensure you have Python installed, then execute the file via your terminal:
```bash
python "HIIT Custom Sprint Tracker.py"
```

## 🧠 Why I Built This
I engineered this custom script to optimize my high-intensity soccer conditioning sessions. Instead of relying on generic clock apps, this terminal engine helps me maintain strict 30-second interval discipline during sprint training to build elite athletic stamina.
